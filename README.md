## 🚀 About Me

> *"The best way to predict the future is to build it."*

I'm **K. Murali Krishna**, a passionate student diving deep into the worlds of **Embedded Systems**, **VLSI Design**, and **Python Development**. I love turning ideas into real, functional hardware and software solutions.

- 🎓 **Currently:** Student & eager learner in the tech ecosystem
- 🔌 **Passion:** Building things that interact with the physical world — circuits, chips, and code
- 🧠 **Currently Exploring:** VLSI design flows, microcontroller programming & Python automation
- ⚡ **Fun Fact:** I find debugging hardware more satisfying than debugging software — both are 🔥 though!
- 🌱 **Goal:** Bridge the gap between silicon and software
